# Assets del Proyecto

Esta carpeta contiene recursos estáticos como imágenes, banners, etc.

## Imágenes

- `discordbanner.jpg`: Banner para el comando !discord con degradado morado

## Cómo actualizar imágenes

1. Sube tu imagen `discordbanner.jpg` a `assets/images/`
2. Reinicia el servidor para que tome la nueva imagen
3. El comando `!discord` automáticamente usará la nueva imagen

## Especificaciones del Banner

- **Nombre del archivo**: `discordbanner.jpg`
- **Ubicación**: `assets/images/discordbanner.jpg`
- **Formato**: JPG (recomendado para banners con degradados)
- **Tamaño recomendado**: 800x200px
- **Peso máximo**: 2MB
- **Contenido**: Degradado morado de abajo hacia arriba

## URL de acceso

La imagen estará disponible en:
- Desarrollo: `http://localhost:3000/assets/images/discordbanner.jpg`
- Producción: `https://tu-dominio.com/assets/images/discordbanner.jpg`

## Diseño sugerido

- **Colores**: Degradado morado (#5865F2 → #4C51BF → #3730A3)
- **Dirección**: De abajo hacia arriba
- **Texto opcional**: Puedes agregar texto o iconos encima del degradado
